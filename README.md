# 프로그램 설치 및 환경 확인
<파이썬 3.10 이상>  
3.14 이상은 pip 오류 발생으로, 낮은버전으로 재설치   
  
python --version (WIN)  
python3 --version (MAC)  

pip --version (WIN)  
pip3 --version (MAC)  

<node.js 18.0 이상>  
node --version  
npm --version  
npx --version  

버전이 모두 출력되어야 정상
